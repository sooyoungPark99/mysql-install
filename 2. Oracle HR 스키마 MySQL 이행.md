# Oracle HR 스키마를 MySQL로 이행 (Python 기반)

Oracle 19c HR 스키마를 MySQL 8.0으로 이행하는 가이드입니다.
Python(cx_Oracle + mysql-connector)으로 DDL 생성, 데이터 이행, 검증까지 전 과정을 다룹니다.

---

### 1. 실습 환경

| 항목 | 값 |
|------|-----|
| OS | Oracle Linux 7.9, UEK 5.4.17 |
| Oracle | 19c EE, SID=orcl (192.168.23.67:1521) |
| MySQL | 8.0 |
| 원본 스키마 | HR (hr/hr) |
| 타겟 DB | MySQL hr |
| 이행 방식 | Python 3 (cx_Oracle + mysql-connector-python) |

---

### 2. 작업 계정 구분

| 구간 | 계정 | 내용 |
|------|------|------|
| 사전 확인 / DDL 생성 | oracle | Oracle Data Dictionary 조회 및 DDL 생성 |
| MySQL 준비 / DDL 적용 | root | MySQL DB 생성, pip 설치, DDL 적용 |
| 데이터 이행 / 검증 | oracle | 데이터 이행 및 건수 검증 |

---

### 3. Oracle → MySQL 타입 매핑

| Oracle 타입 | 예시 | MySQL 타입 | 대상 컬럼 |
|------------|------|-----------|---------|
| NUMBER(p,0) | NUMBER(6,0) | INT | employee_id, department_id 등 |
| NUMBER(p,s>0) | NUMBER(8,2) | DECIMAL(p,s) | salary, commission_pct |
| NUMBER(제한 없음) | region_id | DECIMAL(38,10) | region_id 등 |
| VARCHAR2(n) | VARCHAR2(25) | VARCHAR(n) | 문자열 전반 |
| CHAR(n) | CHAR(2) | CHAR(n) | country_id |
| DATE | hire_date | DATETIME | 날짜 컬럼 |

### 5. Oracle HR 스키마 구조 사전 확인 (oracle)

```bash
su - oracle
sqlplus hr/hr@192.168.23.67:1521/orcl
```

테이블 목록을 확인한다.

``sql`
select table_name from user_tables;
```
``sql
-- 컬럼 정보
COL table_name  FOR A15
COL column_name FOR A25
COL data_type   FOR A15
SET LINESIZE 120

SELECT table_name, column_name, data_type,
       data_precision, data_scale, char_length, nullable
FROM   all_tab_columns
WHERE  owner = 'HR'
ORDER  BY table_name, column_id;

-- 제약 조건
COL table_name       FOR A15
COL constraint_name  FOR A25
COL constraint_type  FOR A5
COL r_constraint_name FOR A25

SELECT table_name, constraint_name, constraint_type, r_constraint_name
FROM   all_constraints
WHERE  owner = 'HR'
  AND  constraint_type IN ('P','U','R')
ORDER  BY table_name, constraint_type;

-- 시퀀스
COL sequence_name FOR A30

SELECT sequence_name, last_number
FROM   all_sequences
WHERE  sequence_owner = 'HR';

-- 뷰
COL view_name FOR A30

SELECT view_name
FROM   all_views
WHERE  owner = 'HR';

-- 트리거
COL trigger_name FOR A30
COL status       FOR A10

SELECT trigger_name, status
FROM   all_triggers
WHERE  owner = 'HR';

EXIT;
```

### 10. 이행 결과 검증 (oracle)

Python 검증 스크립트로 테이블별 건수를 비교한다.

```bash
python3 ~/03_verify.py
```

MySQL에서 SQL로 직접 건수를 비교한다.

```bash
cat > /home/oracle/03_hr_verify.sql << 'EOF'
SELECT 'regions'     AS tbl, COUNT(*) AS cnt FROM regions
UNION ALL SELECT 'countries',   COUNT(*) FROM countries
UNION ALL SELECT 'locations',   COUNT(*) FROM locations
UNION ALL SELECT 'departments', COUNT(*) FROM departments
UNION ALL SELECT 'jobs',        COUNT(*) FROM jobs
UNION ALL SELECT 'employees',   COUNT(*) FROM employees
UNION ALL SELECT 'job_history', COUNT(*) FROM job_history;
EOF

mysql -u root -p hr < 03_hr_verify.sql
```
